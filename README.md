# MatchCast

AI-powered audio match analysis for the Premier League. Pick any finished EPL 2024/25 fixture and get a pundit-style verdict ‚Äî generated by GPT-4o and delivered as audio commentary by ElevenLabs.

**Live:** https://matchcast.vercel.app

---

## What it does

1. Browse all 380 finished EPL 2024/25 fixtures, grouped by date
2. Click **MatchCast** on any match
3. The backend fetches detailed match stats from api-sports.io, preprocesses them into structured signals, generates a pundit-style analysis with GPT-4o, and converts it to audio via ElevenLabs TTS
4. Audio auto-plays in the browser ‚Äî toggle the transcript if you want to read along
5. Results are cached ‚Äî second click is instant, no repeated API spend

---

## Tech Stack

**Backend** ‚Äî Node.js ¬∑ Express 5 ¬∑ TypeScript
**Frontend** ‚Äî React 19 ¬∑ Vite ¬∑ TypeScript ¬∑ TanStack Query ¬∑ Axios
**AI** ‚Äî OpenAI GPT-4o ¬∑ ElevenLabs eleven_multilingual_v2
**Data** ‚Äî api-sports.io (fixture detail) ¬∑ seeded `fixtures.json` (match list)
**Auth** ‚Äî JWT (access token in memory + refresh token in HTTP-only cookie)
**Deployment** ‚Äî Railway (backend) ¬∑ Vercel (frontend)

## Local Development

### Prerequisites

- Node.js 20+
- API keys: OpenAI, ElevenLabs, api-sports.io (only needed for fixture detail ‚Äî match list is served from `src/data/fixtures.json`)

### Backend

```bash
cd backend
cp .env.example .env   # fill in your keys
npm install
npm run dev            # http://localhost:3000
```

### Frontend

```bash
cd frontend
cp .env.example .env   # set VITE_API_BASE_URL=http://localhost:3000
npm install
npm run dev            # http://localhost:5173
```

## API Endpoints

| Method | Path | Auth | Description |
|--------|------|------|-------------|
| GET | `/api/health` | No | Health check |
| POST | `/api/auth/token` | No | Issue anonymous session tokens |
| POST | `/api/auth/refresh` | No | Refresh access token via HTTP-only cookie |
| GET | `/api/matches` | Yes | List finished EPL fixtures (`?limit=&offset=`) |
| GET | `/api/analysis/:fixtureId` | Yes | Generate or retrieve cached match analysis |
