# MatchCast

AI-powered audio match analysis for the Premier League. Pick any finished EPL 2024/25 fixture and get a pundit-style verdict вЂ” generated by GPT-4o and delivered as audio commentary by ElevenLabs.

**Live:** https://matchcast.vercel.app

---

## What it does

1. Browse all 380 finished EPL 2024/25 fixtures, grouped by date
2. Click **MatchCast** on any match
3. The backend fetches detailed match stats from api-sports.io, preprocesses them into structured signals, generates a pundit-style analysis with GPT-4o, and converts it to audio via ElevenLabs TTS
4. Audio auto-plays in the browser вЂ” toggle the transcript if you want to read along
5. Results are cached вЂ” second click is instant, no repeated API spend

---

## Tech Stack

**Backend** вЂ” Node.js В· Express 5 В· TypeScript
**Frontend** вЂ” React 19 В· Vite В· TypeScript В· TanStack Query В· Axios
**AI** вЂ” OpenAI GPT-4o В· ElevenLabs eleven_multilingual_v2
**Data** вЂ” api-sports.io (fixture detail) В· seeded `fixtures.json` (match list)
**Auth** вЂ” JWT (access token in memory + refresh token in HTTP-only cookie)
**Deployment** вЂ” Railway (backend) В· Vercel (frontend)

## Local Development

### Prerequisites

- Node.js 20+ (or Docker)
- API keys: OpenAI, ElevenLabs, api-sports.io (only needed for fixture detail вЂ” match list is served from `src/data/fixtures.json`)

### With Docker (recommended)

```bash
cp backend/.env.example backend/.env   # fill in your keys
docker compose up --build
```

- Frontend: http://localhost:80
- Backend: http://localhost:3000

### Without Docker

**Backend**
```bash
cd backend
cp .env.example .env   # fill in your keys
npm install
npm run dev            # http://localhost:3000
```

**Frontend**
```bash
cd frontend
cp .env.example .env   # set VITE_API_BASE_URL=http://localhost:3000
npm install
npm run dev            # http://localhost:5173
```

## API Endpoints

| Method | Path | Auth | Description |
|--------|------|------|-------------|
| GET | `/api/health` | No | Health check |
| POST | `/api/auth/token` | No | Issue anonymous session tokens |
| POST | `/api/auth/refresh` | No | Refresh access token via HTTP-only cookie |
| GET | `/api/matches` | Yes | List finished EPL fixtures (`?limit=&offset=`) |
| GET | `/api/analysis/:fixtureId` | Yes | Generate or retrieve cached match analysis |
